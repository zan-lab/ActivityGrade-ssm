# ActivityGrade-ssm活动评分小程序

已经进行2.0版本重构，[项目地址:ActivityGrade2.0](https://github.com/GenjiLemon/ActivityGrade2.0)

## 项目简介

本项目为活动评分系统小程序的后端，采用SSM架构，后端采用sql数据库。

## 运行条件

* 对应前端支持
* MySQL5.6
* JDK8+
* Tomcat9.0
* 七牛云OSS



## 运行说明

* 前端部署正常
* mvn install成功
* 打包部署Tomcat
* 配置https



## 技术架构

采用Spring+SpringMVC+MyBatis架构
